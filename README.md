## ✨ Key Features

###  User Authentication & Profile
* **Secure Auth:** Registration and Login using Laravel Sanctum tokens.
* **Profile Management:** Users can upload profile pictures during registration.
* **Guest Access:** Public feed is visible to guests, but interactions are protected.

###  Post Management (CRUD)
* **Create Posts:** Rich text content with Image Uploads.
* **Edit/Delete:** Users can strictly manage only their own posts.
* **Visibility Control:** Option to mark posts as **Public** or **Private**.
* **Tagging System:** Organize posts with dynamic hashtags (e.g., #tech, #life).

###  Engagement & Interactions
* **Like System:** Real-time heart/like counter.
* **Bookmarking:** Save posts to a personal "Saved" list.
* **Comments:** Nested commenting system for discussions.
* **Search:** Real-time search filter for post titles and content.

##  Tech Stack

**Backend:**
* Laravel 10/11 (PHP Framework)
* MySQL (Database)
* Laravel Sanctum (API Authentication)

**Frontend:**
* Vue.js 3 (Composition API)
* Vite (Build Tool)
* Axios (HTTP Client)
* Vue Router (Navigation)
